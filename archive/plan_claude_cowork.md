# Plan Thực Hành Claude Cowork — 30 phút/ngày

---

## PHASE 0 — Setup (Ngày 1, ~30 phút)

**Mục tiêu:** Cài đặt xong, chạy được task đầu tiên

| Việc cần làm | Thời gian |
|---|---|
| Đăng ký / nâng cấp tài khoản Claude (cần plan Pro hoặc Team) | 5 phút |
| Tải và cài Claude desktop app tại claude.ai/download | 5 phút |
| Bật tính năng Cowork trong app (Settings → Enable Cowork) | 5 phút |
| Làm task đầu tiên: "Hãy đọc Desktop của tôi và liệt kê 5 file được sửa gần nhất" | 15 phút |

---

## PHASE 1 — Quản lý File & Tài liệu (Ngày 2–7)

> 30 phút/ngày, mỗi ngày 1 bài tập cụ thể

| Ngày | Bài tập |
|---|---|
| Ngày 2 | Yêu cầu Cowork đọc 1 folder dự án → tóm tắt cấu trúc |
| Ngày 3 | Cho Cowork đọc 1 file Word/PDF → trích xuất ý chính ra file mới |
| Ngày 4 | Cho Cowork tổ chức lại folder Downloads (phân loại theo loại file) |
| Ngày 5 | Yêu cầu tạo báo cáo từ nhiều file dữ liệu rời rạc |
| Ngày 6 | Thử task có nhiều bước: "Đọc file A, so sánh với file B, ghi sự khác biệt ra file C" |
| Ngày 7 | Tổng kết: tự ra 1 task thực tế từ công việc của bạn |

---

## PHASE 2 — Tự động hóa Workflow (Ngày 8–14)

> Tập dùng Routines — tác vụ chạy tự động theo lịch

| Ngày | Bài tập |
|---|---|
| Ngày 8 | Tạo Routine đầu tiên: "Mỗi sáng 8h, tóm tắt email chưa đọc ra 1 file" |
| Ngày 9 | Tạo Routine: "Cuối ngày, liệt kê file nào được chỉnh sửa hôm nay" |
| Ngày 10 | Kết hợp Cowork + Slack: tự động post tóm tắt vào channel |
| Ngày 11 | Automation cho dự án code: "Đọc file README + package.json → tạo báo cáo dự án" |
| Ngày 12 | Tạo workflow xử lý dữ liệu lặp lại (ví dụ: chuyển CSV → báo cáo) |
| Ngày 13 | Tối ưu lại Routine đã tạo, thêm điều kiện phức tạp hơn |
| Ngày 14 | Tổng kết: xây 1 workflow hoàn chỉnh cho công việc thực tế |

---

## PHASE 3 — Ứng dụng vào công việc thực tế (Ngày 15–21)

> Áp dụng vào dự án thật, không còn bài tập mẫu

| Ngày | Hướng làm |
|---|---|
| Ngày 15–16 | Xác định 3 tác vụ lặp lại nhất trong tuần → giao cho Cowork |
| Ngày 17–18 | Xây pipeline: input → Cowork xử lý → output chuẩn |
| Ngày 19–20 | Tích hợp với tool đang dùng (VS Code, Notion, Excel...) |
| Ngày 21 | Review: so sánh năng suất trước và sau khi dùng Cowork |

---

## Nguyên tắc luyện tập hiệu quả

- **Bắt đầu bằng task thật** — đừng dùng task giả để luyện
- **Tăng độ phức tạp dần** — 1 bước → nhiều bước → có điều kiện
- **Ghi lại prompt hiệu quả** — những cách ra lệnh nào cho kết quả tốt
- **Đừng làm thay Cowork** — nếu bạn đang tự click/copy thì sai rồi
